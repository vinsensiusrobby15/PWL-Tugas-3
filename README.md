### How To Run?

##### 1. Clone this repository

```
git clone https://github.com/vinsensiusrobby15/PWL-Tugas-3.git
```

##### 2. Switch to direktori

```
cd PWL-Tugas-3
```

##### 3. Collect depedencies

```
npm i
```

##### 4. Run on dev mode

```
npm start
```

##### 5. Run some test suits

```
npm test
```
